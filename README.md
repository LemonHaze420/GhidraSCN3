# GhidraSCN3
A work-in-progress Scenario Script v3 (d3t) processor module for Ghidra

## Usage
Copy into `Ghidra\Processors\SCN3`. 

Build and run with `..\..\..\support\sleigh.bat data\languages\SCN3.slaspec && ..\..\..\ghidraRun.bat`.
